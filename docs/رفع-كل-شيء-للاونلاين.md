# رفع كل التعديلات للأونلاين (كود + بيانات كاملة)

ليس عملات أو فروع فقط — **كل** ما في قاعدة المحلي: عملاء، أصناف، فواتير، إعدادات، عملات، فروع، …

---

## على جهازك — 3 خطوات

### 1) افتح CMD والصق:

```bat
cd /d "d:\erp projects\first click"
scripts\publish-all-to-online.bat
```

- اضغط **Y** للمتابعة  
- اضغط **Y** لرفع الكود (أو **N** إذا رفعته مسبقاً)  
- انتظر تصدير قاعدة البيانات  

### 2) ارفع ملف SQL

سيفتح مجلد `scripts\backups\` — ارفع **أحدث** ملف `backup_....sql` عبر:

**Hostinger → VPS → File Manager**

إلى المسار:

```
/tmp/db_backup.sql
```

(يجب أن يكون الاسم بالضبط `db_backup.sql`)

### 3) على السيرفر — Terminal

```bash
bash /var/www/erp/deploy/publish-all-online.sh
```

### 4) المتصفح

افتح https://firstclickerp.top واضغط **Ctrl+Shift+R**

---

## ماذا يحدث؟

| الخطوة | ماذا تُنقل |
|--------|------------|
| git push | كل تعديلات البرنامج (شاشات، إصلاحات، …) |
| db_backup.sql | كل جداول قاعدة البيانات من المحلي |
| publish-all-online.sh | يبني الموقع ويستورد القاعدة |

---

## تحذير

بعد الاستيراد، **بيانات الموقع الأونلاين = نسخة من جهازك**.

إن كان على الإنتاج فواتير أو عملاء ليسوا على المحلي، **ستُستبدل** بما عندك محلياً.

---

## إذا أردت الكود فقط (بدون مسح بيانات الإنتاج)

```bat
scripts\publish-to-online.bat
```

اختر **1**، ثم على السيرفر:

```bash
bash /var/www/erp/deploy/publish-online.sh
```

**بدون** رفع ملف `db_backup.sql`.
