## متى تختار النسخة الكاملة؟

فقط إذا أردت أن يصبح **موقع الإنتاج = نسخة قاعدة بياناتك المحلية بالكامل**.

| # | أين | ماذا |
|---|-----|------|
| 1 | جهازك | `scripts\رفع-كل-شيء.bat` (تصدير + push `deploy/db_backup.sql`) |

⚠️ **يستبدل كل بيانات الإنتاج** (فواتير، إعدادات التواصل، صفحة الدخول) بنسخة المحلي.

## إذا ظهر «فشل الاتصال بالخادم» أو رجعت بيانات التواصل الافتراضية

1. غالباً **الـ API متوقف** (Redis / nginx / PHP) — وليس أن الإعدادات حُذفت.
2. على السيرفر **بدون استيراد قاعدة**:

```bash
bash /var/www/erp/deploy/repair-api-quick.sh
```

3. حدّث المتصفح: **Ctrl+Shift+R**
