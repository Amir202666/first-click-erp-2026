أيقونة الموقع (Favicon) - First Click ERP
==========================================

لاستبدال الأيقونة الافتراضية بلوغو First Click:

1. جهّز ملف اللوغو بأحد الشكلين:
   - favicon.ico (مثلاً 32×32 أو 16×16) للتوافق مع المتصفحات
   - favicon.svg (أو صورة PNG صغيرة)

2. ضع الملف في هذا المجلد (public):
   - frontend/public/favicon.ico
   و/أو
   - frontend/public/favicon.svg

3. الصفحة الرئيسية (index.html) تشير بالفعل إلى:
   - /favicon.ico
   - /favicon.svg (كبديل)

4. بعد إضافة الملف، أعد تحميل الصفحة أو امسح الكاش لرؤية الأيقونة الجديدة.
